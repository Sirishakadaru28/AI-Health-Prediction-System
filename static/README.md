# AI Health Prediction Application

## Project Overview

This is a Python based AI Health Prediction Application developed for patient health analysis.

The application predicts possible health conditions using patient blood test values such as:

- Glucose Level
- Haemoglobin
- Cholesterol

A Machine Learning model analyzes the data and generates health risk predictions.

## Technologies Used

- Python
- Flask Framework
- SQLite Database
- HTML
- CSS
- Bootstrap
- Machine Learning
- Random Forest Algorithm


## Features

- Add Patient Details
- View Patient Records
- Update Patient Information
- Delete Patient Records
- AI Based Health Prediction
- Input Validation
- Persistent Database Storage


## Machine Learning

A Random Forest Classifier model is trained using healthcare parameters.

Input Features:
- Glucose
- Haemoglobin
- Cholesterol

Output:
- Health Risk Prediction


## How to Run

Install dependencies

pip install -r requirements.txt


Train Model

python train_model.py


Run Application

python app.py


Open Browser:

http://127.0.0.1:5000
